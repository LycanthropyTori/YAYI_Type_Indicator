# YYTI — 压抑 Type Indicator

## What This Is

一款趣味心理测试，模仿 SBTI 算法框架，测量你在学业、职场、家庭、亲密关系、内心五个生活域中的"压抑指数"，生成 15 位模式串，匹配 25 种预定义人格类型。

核心玩法：3 选 1 答题 → 15 维评分 → 模式匹配 → 人格卡片 + 维度解析。

## Core Value

纯前端单 HTML 文件，零依赖，数据全内嵌。拿来即用，无服务端。

## 算法参考（SBTI）

- 15 个维度，每维 2 题 × 3 选项（1-3 分），原始分 2-6
- 评分：`≤3=L`, `4=M`, `≥5=H` → 拼成 15 位模式（如 `HHH-HMH-MHH-HHH-MHM`）
- 预定义 25 种类型 pattern，计算欧几里得距离 + 精确命中数
- 匹配度 < 60% → 兜底人格
- 特殊触发题 → 隐藏人格

## 人格类型库（25 种预定义 + 2 种隐藏）

**学业组：** 学者、学酥、学渣、学历焦虑者、学术混子
**职场组：** 卷王、摸鱼大师、996幸存者、职场透明人、晋升机器
**家庭组：** 家族担当者、家庭叛逆者、家庭和事佬、离家出走者、妈宝/爸宝
**亲密组：** 恋爱脑、分手专家、独行玩家、情感乞丐、海王/海后
**内心组：** 压抑容器、定时炸弹、行尸走肉、慢性崩溃、情绪海绵
**隐藏组：** 麻木者、爆发者（特殊触发）

## 特殊机制

- **隐藏人格门**：特殊题目（如"你平时有什么爱好"→ 饮酒选项）触发隐藏人格
- **兜底人格**：匹配度 < 60% 触发兜底人格

## 技术实现

- 纯 HTML 单文件，内嵌 CSS + JS
- 3 个屏幕：首页 → 测试页 → 结果页
- 题目随机打乱，3 选 1
- 结果页：类型卡片 + 15 维度评分详情
- 响应式设计，移动端友好
- 风格参考 SBTI：植物绿配色，圆角卡片，柔和渐变

## Out of Scope

- 服务端 / 数据库
- 多语言
- 结果分享功能
- 答题记录保存
