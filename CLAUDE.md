# YYTI — 压抑 Type Indicator

## Project

YYTI — 压抑 Type Indicator：趣味心理测试，测量学业/职场/家庭/亲密关系/内心五个域的压抑指数，生成15位模式串匹配25种人格类型。

## Core Value

纯前端单 HTML 文件，零依赖，数据全内嵌。拿来即测。

## Technical

- **Stack**: 纯 HTML + CSS + JS（单文件）
- **Algorithm**: SBTI-style 15维模式匹配
- **Dimensions**: 5模型 × 3维度 = 15维
- **Types**: 25种预定义人格 + 2种隐藏人格
